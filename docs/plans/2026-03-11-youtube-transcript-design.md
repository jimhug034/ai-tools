# YouTube 视频转录工具 - 设计文档

**日期**: 2026-03-11
**状态**: 已批准

## 1. 项目概述

构建一个类似 [lynote.ai](https://lynote.ai/zh/youtube-transcript) 的 YouTube 视频转录工具，提供视频转文字、AI 摘要、翻译和导出功能。

### 核心目标

- 即时生成 YouTube 视频的准确转录
- AI 驱动的智能摘要和翻译
- 极简的用户体验
- 完全免费，无需注册

## 2. 技术选型

| ------------ | ----------------------------------------------- | ------------------------------- |

| **图标**     | lucide-react                                    | 轻量级图标库                    |
| **转录 API** | youtube-transcript (优先) + Groq Whisper (备用) | 混合方案，速度与覆盖兼顾        |
| **AI 服务**  | Groq Llama 3                                    | 免费层慷慨，速度极快            |
| **部署**     | Vercel                                          | Next.js 原生支持，全球 CDN      |
| **测试**     | Vitest + agents-browser                         | 单元测试 + E2E 测试             |

## 3. 架构设计

```
┌─────────────────────────────────────────────────────────────────┐
│                           用户浏览器                              │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────┐       │
│  │ 输入 YouTube │ → │   展示结果   │    │   导出功能   │       │
│  │     URL      │    │ (转录/摘要)  │    │  (TXT/SRT)   │       │
│  └──────────────┘    └──────────────┘    └──────────────┘       │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                      Next.js API Routes                          │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │  /api/transcript  → 获取 YouTube 转录                     │   │
│  │  /api/summarize   → AI 生成摘要                          │   │
│  │  /api/translate   → 翻译转录文本                          │   │
│  └──────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                          外部服务                                 │
│  ┌──────────────────┐    ┌──────────────────┐                   │
│  │youtube-transcript│    │   Groq API       │                   │
│  │   (免费字幕)     │    │ (Llama 3 摘要)   │                   │
│  └──────────────────┘    └──────────────────┘                   │
└─────────────────────────────────────────────────────────────────┘
```

### 核心流程

1. 用户输入 YouTube URL
2. 验证 URL 格式
3. 优先使用 `youtube-transcript` 获取字幕
4. 如无字幕，使用 Groq Whisper API（免费层）
5. 使用 Groq Llama 3 生成摘要和翻译
6. 结果展示，支持导出

## 5. 数据流与状态管理

### 请求流程

```
用户输入 URL
     ↓
验证 URL 格式
     ↓
POST /api/transcript
     ↓
1. 尝试 youtube-transcript (免费)
2. 失败 → 使用 Groq Whisper API
3. 返回 { transcript, meta }
     ↓
展示转录文本
     ↓
用户点击"生成摘要"
     ↓
POST /api/summarize
     ↓
1. 调用 Groq Llama 3
2. 返回摘要文本
     ↓
用户选择翻译语言
     ↓
POST /api/translate
     ↓
1. 调用 Groq Llama 3 翻译
2. 返回翻译结果
```

### 状态结构

```typescript
interface AppState {
  url: string; // 输入的 URL
  loading: boolean; // 加载状态
  transcript: Transcript | null; // 转录结果
  summary: string | null; // AI 摘要
  translation: string | null; // 翻译结果
  targetLang: string; // 目标翻译语言
  error: string | null; // 错误信息
}
```

## 6. 错误处理

### 需要处理的场景

| 场景                   | 处理方式                                                |
| ---------------------- | ------------------------------------------------------- |
| **无效的 YouTube URL** | 实时验证，提示"请输入有效的 YouTube 链接"               |
| **视频无字幕**         | 尝试 Groq Whisper API；如失败，提示"该视频暂无可用字幕" |
| **视频过长（>2小时）** | 提示"视频较长，处理可能需要较长时间"                    |
| **API 限流**           | 提示"请求过于频繁，请稍后再试"                          |
| **网络错误**           | 友好提示，提供重试按钮                                  |
| **导出失败**           | 提示"导出失败，请重试"                                  |

### 用户提示信息（双语）

| 场景     | 中文                         | English                                 |
| -------- | ---------------------------- | --------------------------------------- |
| 正在处理 | "正在获取转录，请稍候..."    | "Fetching transcript, please wait..."   |
| 处理成功 | "转录获取成功！"             | "Transcript fetched successfully!"      |
| 无字幕   | "该视频暂无可用字幕"         | "No subtitles available for this video" |
| API 错误 | "服务暂时不可用，请稍后重试" | "Service temporarily unavailable"       |
